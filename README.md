# Student-Data-Management-System

## Description
This is a PHP program to manage student records using arrays, functions, and date calculations.

## Features
- Stores details of 5 students using an associative array
- Calculates average marks
- Determines grade based on marks
- Calculates age from date of birth
- Displays all student details in tabular format

## Technologies Used
- PHP
- HTML
- CSS

## Output
The program displays:
- Student Name
- Marks
- Date of Birth
- Age
- Grade
- Average Marks

## How to Run
1. Place the file inside the XAMPP `htdocs` folder
2. Start Apache in XAMPP
3. Open the browser and run:
   `http://localhost/student_project/student_management.php`
